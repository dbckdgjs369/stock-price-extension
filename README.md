# Samsung Stock Widget

현재 웹페이지 위에 삼성전자 시세 카드를 고정해서 보여주는 Chrome 확장입니다.

## 동작

- 아무 웹페이지에서 `Shift + 드래그`로 위젯 위치와 크기를 지정
- 또는 팝업의 `위치 다시 잡기`를 누른 뒤 한 번 드래그
- 선택한 영역에 삼성전자(`005930.KS`) 시세 카드 표시
- 약 1분마다 시세 자동 갱신
- 팝업에서 위젯 표시 여부와 수동 새로고침 제어

## 설치

1. Chrome에서 `chrome://extensions/` 열기
2. 우측 상단 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램 로드` 클릭
4. 이 폴더를 선택

## 사용

1. 구글 같은 일반 웹페이지 열기
2. `Shift` 키를 누른 채 마우스로 드래그
3. 드래그한 위치에 시세 카드가 생성됨
4. 다시 위치를 바꾸고 싶으면 같은 방식으로 다시 드래그

## 참고

- 시세 데이터는 Yahoo Finance 응답을 사용합니다.
- 무료 데이터 특성상 완전한 체결 실시간이 아니라 지연 시세일 수 있습니다.
- `chrome://` 같은 브라우저 내부 페이지에서는 동작하지 않습니다.
