# Stock Price Widget

현재 웹페이지 위에 여러 주식/코인 시세 위젯을 배치하는 Chrome 확장입니다.

## 동작

- 팝업에서 주식/코인 모드를 고르고 종목명 또는 티커를 검색
- 검색 결과에서 `생성` 클릭
- 웹페이지로 돌아가 `Shift + 드래그`로 새 위젯 생성
- 같은 방식으로 여러 종목 위젯 추가 가능
- 기존 위젯은 `Shift + 드래그`로 이동 가능
- 기존 위젯 우하단 핸들은 `Shift + 드래그`로 크기 조절 가능
- 팝업에서 현재 생성된 위젯 목록 확인 및 삭제 가능

## 설치

1. Chrome에서 `chrome://extensions/` 열기
2. `개발자 모드` 활성화
3. `압축해제된 확장 프로그램 로드` 클릭
4. 이 폴더 선택

## 참고

- 시세 및 검색 데이터는 Yahoo Finance 응답을 사용합니다.
- 무료 데이터 특성상 실시간 체결가가 아니라 지연 시세일 수 있습니다.
- `chrome://` 같은 브라우저 내부 페이지에서는 동작하지 않습니다.
